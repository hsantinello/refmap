Como usar o Ref Map

Em três passos você vai da imagem ao prompt pronto para usar no seu gerador de IA.

1
Importe suas imagens de referência
Arraste imagens diretamente para o canvas, ou clique em + Importar na barra superior.
O app aceita PNG, JPG e WEBP gerados por qualquer ferramenta de IA — ComfyUI, Midjourney, A1111, FLUX e outras.
2
Clique em uma imagem para ver seus metadados
Ao selecionar uma imagem, um painel aparece ao lado com todas as informações extraídas automaticamente: modelo, sampler, steps, seed, LoRAs e tags de estilo.
Para imagens sem metadados embutidos, o app usa IA de visão para gerar as tags automaticamente.
3
Monte seu prompt no Prompt Builder
Clique em qualquer tag para adicioná-la ao Prompt Builder na parte inferior da tela.
Reordene os chips arrastando, edite o texto manualmente, e use ✦ Otimizar para que a IA reescreva o prompt no formato ideal para o seu gerador. Quando estiver pronto, clique em Copiar.
Organizando o canvas
Grupos — selecione várias imagens e agrupe por tema. Arrastar o grupo move todas as imagens juntas.
Múltiplos canvas — crie canvas separados para projetos diferentes pela barra de abas no topo.
Renomear canvas — duplo clique na aba para renomear.
Prompt Builder
Histórico — os últimos 10 prompts ficam salvos. Clique no ícone de relógio para acessar.
Campo de texto livre — escreva partes do prompt manualmente junto com as tags.
Otimizar — escolha o modelo de destino (FLUX, Midjourney, SD...) e a IA adapta o prompt.

Combine referências de imagens diferentes — selecione tags de várias imagens no mesmo prompt builder para misturar estilos.
A ordem dos chips importa — arraste para reordenar. Tags no início do prompt têm mais peso na maioria dos geradores.
Use "On Top" no menu para manter o Ref Map visível enquanto gera imagens em outro app.
Clique em "Configure uma API key" na imagem com erro para ir direto às configurações e conectar sua chave.
